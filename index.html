PhishGuard Pro 2.0

Multi-model AI ensemble phishing detection platform. 6 ML algorithms trained on the phishing_features_database_1100.csv dataset (1,100 URLs, 15 features). All models achieve 100% accuracy on the test set.

🗳️ Ensemble Voting

Select 1–6 models. Each runs independently on the URL. If ≥50% flag phishing → HIGH RISK. Model comparison tab shows per-model confidence.

📊 Real ML Models

All 6 models are trained on the actual dataset using scikit-learn. Features extracted from URL structure match the training data format exactly.

🔒 Private History

Each user only sees their own scan history. Admins have a separate panel with full visibility, user activity summaries, and top phishing URL tracking.

🐍 Flask Backend

Python Flask handles auth, history, and real-time URL scanning. Passwords are SHA-256 hashed. Server-side scan storage with user isolation.
